# Nebula

A social media fullstack project using microservices architecture.

## Usage

``` bash
git clone https://github.com/brandon-a-pinto/nebula.git
cd nebula
make prepare
sudo make up_build
```

## Technologies

- Golang
- TypeScript
- React
- gRPC
- RabbitMQ
- PostgreSQL
- MongoDB
- Docker
